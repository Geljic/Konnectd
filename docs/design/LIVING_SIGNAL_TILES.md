# Living Signal Tiles

Date: 2026-06-11

Konnectd started with a Garden Pop / friendly-face tile style. That gives the game charm, but the modes now need a shared visual system so Groups, Next Steps, and Crossed Signals feel like one product instead of separate prototypes.

## Direction

Use **Living Signal Tiles** as the broad product language.

The app should feel friendly, clever, and original: inspired by the warmth and tile personality of *Threes!*, but not visually copied. The core identity should be reusable across current and future modes.

## Core Rules

- Keep one shared base tile shape across all game modes.
- Use dark navy tile bodies in dark mode, with theme-driven rims, caps, pips, or signal accents.
- Faces are allowed, but they should be a configurable personality layer rather than the whole brand.
- Avoid making every mode equally face-heavy.
- Crossed Signals should feel alive through signal dots, receivers, row/column tabs, and connecting lines, not full cartoon faces.
- Garden Pop can remain a theme pack, but the whole product should not be locked into a garden concept.

## Mode Treatment

### Groups

Groups can be the most expressive mode.

- Full friendly tile faces are appropriate.
- Solved groups should use warm category bands.
- The game can carry the most personality here because the board is simple.

### Next Steps

Next Steps should use the same tile body but shift the emphasis toward order.

- Keep faces or eyes, but quieter than Groups.
- Add ordered pips, path accents, or thread-like connection details.
- The mode should feel like arranging a living sequence, not a different tile game.

### Crossed Signals

Crossed Signals should be the most logic-forward mode.

- Avoid full tile faces by default.
- Use small eyes, dots, or receiver marks to keep the tiles friendly.
- Row and column clues should use colored signal tabs and subtle connection lines.
- Selected or solved rows/columns can use the same theme colors as solved Groups.

## Settings And Monetization

The current settings model can become a useful cosmetics system:

- **Board theme:** Classic, Garden Pop, future seasonal or premium palettes.
- **Tile personality:** Subtle, Peek, Bold.
- **Mode accents:** signal lines, tile caps, solved strips, row/column tabs.

This keeps cosmetics meaningful without changing puzzle rules.

## Recommendation

Move toward a unified Konnectd tile system before launch. Keep Garden Pop as one theme, but make the brand wider: friendly signal tiles, living logic tiles, and playful puzzle clarity.
